Mobile Auction System

This is a Mobile Auction System for Android-based devices which includes two apps, one for seller, one for buyyer. I developed this app with my friend as a senior design project which won 4th place in Fresh Brain New Ideas Competition in Industrial Category to help buyers with their shopping experience; specifically allowing several stores to bid for the products the customer desires while allowing consumers to find the best combinations of prices from different sellers. There is also location control for the buyyers. They have to be in one of the registered stores to start an auction. Executed solution using IBM ILOG CPLEX Optimizer, created SOAP-based web services, used Java, MySQL, PHP.
Used Html, Css, Js, Bootstrap.
