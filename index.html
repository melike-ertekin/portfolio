Smart Pass and Payment System

Created an Android app tailored for authorization profile level entry into campus buildings and a payment system to use online money in the campus restaurants. Implemented location check using location API, and push notification using Google Cloud Messaging. Created SOAP web services. Used Java, Python, Raspberry Pi, Oracle, and PHP.

Mobile Auction System

Developed a Mobile Auction System as a senior design project to help buyers with their shopping experience; specifically allowing several stores to bid for the products the customer desires while allowing consumers to find the best combinations of prices from different sellers. Implemented push notification using Google Cloud Messaging (GCM). Executed solution using IBM ILOG CPLEX Optimizer, used Java, MySQL, PHP, SOAP web services.
